ก่อนใช้ระบบนี้

คุณรู้สึกแบบนี้ไหม?

ถ้าใช่ — คุณไม่ได้ขี้เกียจ คุณแค่ยังไม่มีระบบ

😵

ตื่นขึ้นมาทุกวัน ไม่รู้ว่าวันนี้ควรทำอะไรก่อน — มีงานเยอะในหัว แต่ไม่รู้จะเริ่มจากตรงไหน เลยทำแต่งานด่วน แทนที่จะทำงานสำคัญ

📋

มีแผนในหัวเยอะมาก แต่ไม่เคยทำสำเร็จสักแผน — ตั้งเป้าหมายทุกต้นปี ทุกต้นเดือน แต่พอสิ้นเดือนก็ลืมไปแล้วว่าตั้งไว้ว่าอะไร

💰

เงินเดือนออก แล้วก็หายไปโดยไม่รู้ไปไหน — ไม่ได้ซื้ออะไรใหญ่โต แต่ทำไมเงินถึงไม่พอ ไม่รู้จะเริ่มออมหรือลงทุนจากตรงไหน

🎬

อยากทำคอนเทนต์ แต่ไม่มีระบบ เลยทำๆ หยุดๆ — มีไอเดียเยอะ แต่พอถึงเวลาทำจริงก็ไม่รู้จะเริ่มยังไง เลยผัดวันไปเรื่อยๆ

⚡

รู้สึกว่าตัวเองยุ่งตลอด แต่ชีวิตไม่ได้ขยับไปข้างหน้า — ทำงานหนักทุกวัน แต่พอมองย้อนกลับก็ไม่รู้ว่าปีที่ผ่านมาได้อะไรจากชีวิตบ้าง

ปัญหาเหล่านี้ไม่ได้เกิดจากความขี้เกียจ

ทุกอย่างที่คุณต้องการ อยู่ในที่เดียว

ครอบคลุมทุกด้านของชีวิตและธุรกิจ — เชื่อมถึงกันอัตโนมัติ ไม่กระจัดกระจาย

SYSTEM 01
🧭
Navigator System

ระบบตั้งเป้าหมายและจัดทิศทางชีวิต — รู้ทันทีว่าวันนี้ต้องทำอะไรก่อน

  • Goals → OKR breakdown ไปถึง Daily Action
  • Projects & Tasks เชื่อมถึงกันอัตโนมัติ
  • Dashboard รายวัน — ไม่ต้องคิดเองทุกเช้า
SYSTEM 02
🎬
Creator Pipeline

ระบบสร้างคอนเทนต์ตั้งแต่ไอเดียจนถึงโพสต์ — ผลิตได้สม่ำเสมอโดยไม่รอแรงบันดาลใจ

  • Idea → Script → Production SOP ในที่เดียว
  • Content Calendar แบบ Visual รวมทุก Platform
  • AI Prompt ช่วย Script ทุก Format
SYSTEM 03
⚙️
Lifestyle Design

ระบบออกแบบชีวิตประจำวัน — ตั้งแต่ Habit Tracker ไปจนถึง Calendar รวมทุกอย่าง

  • Habit Tracker + EXP Gamification System
  • Personal Calendar รวม 7 ระบบในหน้าเดียว
  • Morning / Evening Routine Template
SYSTEM 04
💰
Finance System

ระบบจัดการเงินส่วนตัว — เห็นภาพเงินทั้งหมดในที่เดียว รู้ทันทีว่าเงินไปไหน

  • Income & Expense + 5-Color Category System
  • Budget split อัตโนมัติ (Ops / Savings / Tax)
  • Debt, Subscription & Wishlist Tracker
SYSTEM 05
🏢

  • Customer CRM + Client Project Board
  • Product Catalog + Sales Pipeline
  • Business Cashflow Dashboard + KPI Charts

Investment Intelligence

  • Portfolio Tracker (หุ้น / ETF / Crypto / REIT)
  • AI Research Prompt พร้อมใช้ทันที

ก่อน vs หลัง

ชีวิตเปลี่ยนยังไง

เปรียบเทียบง่ายๆ ว่าชีวิตก่อนและหลังมีระบบต่างกันยังไง

😵 ก่อนมีระบบ
  • ตื่นขึ้นมาไม่รู้จะทำอะไรก่อน
  • เป้าหมายอยู่ในหัว ไม่เคยลงมือจริง
  • เงินหมดทุกเดือนโดยไม่รู้สาเหตุ
  • คอนเทนต์ทำๆ หยุดๆ ไม่ต่อเนื่อง
  • ยุ่งตลอด แต่ชีวิตไม่ขยับไปไหน
  • รู้สึก overwhelmed ทุกวัน
🌟 หลังใช้ North Star OS
  • รู้ทันทีว่าวันนี้ต้องทำอะไรก่อน
  • เป้าหมายถูก breakdown ไปจนถึง daily action
  • เห็นภาพเงินทั้งหมดในที่เดียว
  • คอนเทนต์ไหล มีระบบ ไม่รอแรงบันดาลใจ
  • ทำน้อยลง แต่ได้ผลมากขึ้น
  • ชีวิตมีทิศทาง รู้สึก in control

FAQ

ถ้าไม่เจอคำตอบที่ต้องการ ทักมาถามได้เลยที่ LINE OA

AI Prompts ใช้กับอะไรได้บ้าง?+
ใช้ได้กับ ChatGPT, Claude, Gemini หรือ AI ตัวไหนก็ได้ครับ แค่ copy prompt แล้ววางได้เลย ไม่ต้องมี subscription พิเศษเพิ่ม แค่ Free tier ของ ChatGPT ก็ใช้ได้
ต้องเสียค่า Notion ไหม?+
ไม่ต้องเลยครับ Notion Free Plan ใช้ได้ 100% ไม่มีฟีเจอร์ไหนที่ต้องการ Notion Plus หรือ Pro
ต้องรู้จัก Notion มาก่อนไหม?+
ไม่ต้องครับ มาพร้อม North Star Setup Guide ที่สอนตั้งแต่การ Duplicate Template ยังไง ไปจนถึงการใช้แต่ละระบบ มือใหม่ก็ใช้ได้
ซื้อแล้วได้รับของอะไรบ้าง และยังไง?+
หลังสั่งซื้อผ่าน LINE OA ระบบจะส่ง Duplicate Link + คู่มือให้อัตโนมัติครับ กด Duplicate แล้วใช้งานได้เลย
